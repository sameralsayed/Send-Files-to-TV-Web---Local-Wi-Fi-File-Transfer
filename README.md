# 📡 Send Files to TV Web - Local Wi-Fi File Transfer

**📌 Description**  
A sleek, fully functional, mobile-first web replica of the **Send files to TV** Android app.  
Select any files, scan your local Wi-Fi network, discover Smart TVs / PCs / tablets automatically, and transfer files instantly at full Wi-Fi speed. No internet, no cloud, no accounts.

**✨ Features**  
- 📤 Drag & drop + multi-file selection  
- 🔍 Real-time network scan with device discovery  
- 📱 Beautiful device list (TVs, PCs, tablets)  
- 📊 Live transfer progress with per-file bars  
- 📥 Received files section with instant open  
- 📜 Full transfer history (saved locally)  
- ⚙️ Settings panel (download path + auto-accept)  
- 🌐 100% responsive & works offline after first load  
- 🚀 Simulated high-speed local transfers  

**🏷️ Tags / Keywords**  
`send-files-to-tv`, `local-file-transfer`, `wifi-file-sharing`, `cast-files-to-tv`, `android-tv-file-sender`, `web-file-transfer`, `dlna-alternative`, `local-network-share`, `file-cast-web`, `smart-tv-file-share`, `html5-file-sharing`

**🛠️ How to run**  
1. Download all 4 files into the **same folder**  
2. Open `index.html` in any modern browser  
3. Tap “Scan Network” → select files → send instantly!
